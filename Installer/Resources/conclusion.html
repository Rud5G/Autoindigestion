Autoindigestion

Autoindigestion is installed and will automatically run every morning at 08:00 hours.

Important! In order to download iTunes Connect reports, you must add one or more vendor plist files to /Library/Autoindigestion/Vendors. You can use the ExampleVendor.plist as a guide.

iTunes Connect reports will be downloaded to /Users/Shared/iTunes Connect.

For detailed information about Autoindigestion configuration options, visit the Autoindigestion project site
