Autoindigestion

Autoindigestion is a command line tool for Mac OS X that downloads all available sales reports from iTunes Connect. It uses the Auto-Ingest tool supplied by Apple to download individual reports.

The Autoindigestion tool is open source software created by Able Pear Software. It is available for free under a BSD-style open source license from https://github.com/AblePear/Autoindigestion.

You will be guided through the steps necessary to install or update the Autoindigestion tool on this Mac.
